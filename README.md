# vue-element-uploader
> A Vue.js & element-ui upload and image crop component

## Preview
![demo](/demo.gif)

## Features
* Allow single file or multiple files
* File validation
* Queue management
* Preview a image file
* Crop a image file
* Recover a Cropped file
* Chunk uploads
* Recover upload and Fast file transfer
* Easy to use
